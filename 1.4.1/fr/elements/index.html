    Aperçu des éléments utilisés

    Structure de document

    • <TEI>
    • Zone de texte : <text>
    • Zone de tête : <teiHeader>

    Éléments dans la zone de texte

    Balisage critique du texte

    • Abréviations et leur résolution : <abbr> <expan>
    • Erreur et leur correction : <sic> <corr>
    • Sélection de l'une des deux variantes : <choice>
    • Entrées d'appareil avec lemme et lectures : <app> <lem> <rdg>
    • Ajouts : <add> <addSpan>
    • Suppressions : <del> <delSpan>
    • Substitutions : <subst>
    • Dommages : <damage> <damageSpan>
    • Lacunes et leur supplément par les éditeurs : <gap> <space> <supplied>
    • Changement de main : <handShift>
    • Lecture incertaine : <unclear>
    • Point d'ancrage pour les balises inter-hiérarchies : <anchor>

    Balisage de la structure du texte

    • Zone de texte et zone de commentaire : <body> <back>
    • Sections : <div> <seg>
    • Paragraphes dans l'original : <p>
    • Tableaux : <table> <row> <cell>
    • Listes : <list> <item>
    • Graphiques : <figure> <graphic>
    • Bouleversements :
      • Début de la page : <pb>
      • Début de la colonne : <cb>
      • Début de la ligne : <lb>
    • Autres parties du texte :
      • Rubriques, titres : <head>
      • Signatures: <signed>
      • Citations et discours direct : <quote> <q>
      • Gardiens et plaignants : <fw>
      • Étiquettes : <label>
      • Blocs anonymes : <ab>
      • Notes : <note>
      • Références, informations bibliographiques : <bibl> <ref> <listBibl>
      • Signes de ponctuation traités en fonction de la langue : <pc>
    • Changement de langue : <foreign> <orig>
    • Points forts : <hi>

    Balisage du contenu

    • Dates : <date> <precision>
    • Temps : <time>
    • Nombres et chiffres : <num>
    • Mesures, poids, monnaies : <measure> <measureGrp>
    • Registres :
      • Lemmes ou mots-clé : <term>
      • Personnes : <persName>
      • Organisations : <orgName>
      • Lieux : <placeName>

    Éléments dans la zone de tête

    Description du fichier

    • <fileDesc>
    • Description de la page de titre :
      • Titre et série : <titleStmt> <title> <publicationStmt> <seriesStmt>
      • Responsabilité : <editor>, <publisher> <respStmt> <resp>
      • Licence : <availability> <licence>
    • Description de la source : <sourceDesc> <msDesc>
      • Témoins textuels : <listWit> <witness>
      • Description bibliographique : <msIdentifier> <altIdentifier> <idno> <repository> <settlement>
      • Description du contenu de la source : <msContents> <msItem> <author> <textLang> <docImprint> <pubPlace> <publisher> <summary>
      • Histoire de la source : <history> <origin> <origDate> <origPlace>
      • Informations Complémentaires : <additional> <adminInfo> <custodialHist> <custEvent>
      • Description physique de la source : <physDesc>
        • Reliure : <bindingDesc>
        • Mains : <handDesc> <handNote>
        • Sceaux : <sealDesc> <seal>
        • Matériau : <objectDesc> <supportDesc> <support> <material>
        • Étendue : <extent> <dimensions> <height> <width>
        • Foliation : <foliation>
        • État de conservation : <condition>

    Description du codage

    • <encodingDesc> <editorialDecl>

    Description du profil du texte

    • <profileDesc> <textClass>
    • Mots clés: <keywords> <term>
