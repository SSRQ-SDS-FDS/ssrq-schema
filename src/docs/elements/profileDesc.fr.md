# `<profileDesc/>` (description du profil)

## Description

Contient des éléments pour décrire le texte à l'aide de mots-clés. 

## Modèle de contenu

- **header**: [textClass](textClass.md)

## Exemples

### Exemple 1

Structure modèle

```xml
<egXML xmlns:ns0="http://www.tei-c.org/ns/Examples">
    <profileDesc>
        <textClass>
            <keywords>
                <term ref="key000192" />
                <term ref="key000097" />
                <term ref="key000049" />
                <term ref="key000193" />
            </keywords>
        </textClass>
    </profileDesc>
</egXML>
               
```
