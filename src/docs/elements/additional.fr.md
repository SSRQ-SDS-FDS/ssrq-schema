# `<additional/>` (informations complémentaires)

## Description

Contient des références aux éditions, regesta et littérature.

## Modèle de contenu

- **core**: [listBibl](listBibl.md)

## Exemples

### Exemple 1

Exemple de structure

```xml
<egXML xmlns:ns0="http://www.tei-c.org/ns/Examples">
    <additional>
        <listBibl>
          
        </listBibl>
    </additional>
</egXML>
               
```
