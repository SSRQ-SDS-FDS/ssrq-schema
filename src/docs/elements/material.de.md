# `<material/>`

## Beschreibung

Beschreibt das Trägermaterial.

## Inhaltsmodell

Enhält keine Elemente oder Text.

## Attribute

### @type

Trägermaterial

*Mögliche Werte*

- paper – *Papier*
- parchment – *Pergament*

## Beispiele

### Beispiel 1

Modellhafter Aufbau mit zwingender Verwendung von @type und ohne Textinhalt

```xml
<egXML xmlns:ns0="http://www.tei-c.org/ns/Examples">
    <material type="paper" />
</egXML>
               
```
