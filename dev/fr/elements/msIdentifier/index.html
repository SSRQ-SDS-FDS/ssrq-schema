    <msIdentifier/> (identifiant du manuscrit)

    Description

    Décrit les emplacements actuels et historiques de la source.

    Modèle de contenu

    • header: idno
    • msdescription: altIdentifier repository
    • namesdates: settlement

    Exemples

    Exemple 1

    Exemple d'identifiant de manuscrit

    <egXML xmlns:ns0="http://www.tei-c.org/ns/Examples">
    <msIdentifier>
        <idno source="https://suche.staatsarchiv.djiktzh.ch/detail.aspx?ID=3737536" xml:lang="de">StAZH A III 2, S. 353-355</idno>
        <repository xml:lang="de">Staatsarchiv Zürich</repository>
    </msIdentifier>
</egXML>
